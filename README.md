AI PDF Chatbot

An AI-powered chatbot that allows users to upload PDF documents and interact with them using natural language queries.

Features
- Upload and process PDF documents
- Extract and understand document content
- Ask questions and get context-aware answers
- Semantic search using embeddings

Tech Stack
- LangChain
- OpenAI API
- Node.js
- Next.js (Frontend)
- Vector Database (Supabase)

Description
This project demonstrates how Large Language Models (LLMs) can be used to build an intelligent document question-answering system. It converts PDF content into embeddings and enables users to query documents conversationally.

Setup
1. Clone the repository
2. Install dependencies
3. Add API keys in `.env`
4. Run frontend and backend

Output
Interactive chatbot interface for querying PDF documents.
